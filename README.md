\# 💰 Personalized Finance Tracker



\## 🚀 Features

\- Track income and expenses

\- Category-based expense management

\- Expense prediction system

\- Alerts for overspending

\- Data visualization



\## 🛠 Tech Stack

\- Python

\- PyQT5 (GUI)

\- CSV (data storage)



\## ▶️ How to Run

1\. Clone repo

2\. Run:

&#x20;  python main.py
